CSV Rearranger

Reorder, filter, or duplicate columns in any delimiter-separated data

Configuration

Default: comma (,) — use \t for tab

1-based column indices — repeat to duplicate, omit to drop

Data

Each line is treated as one row

Quick example

Input: Alice,30,Engineer

Delimiter: ,   Column order: 3,1

Output: Engineer,Alice — column 3 first, then 1; column 2 dropped

Data never leaves your browser.
